con una entidad inequívoca; para El Tunco se contrastó el Geographic Names Server y para Coatepeque
el registro de GeoNames. Los valores se almacenan como `[longitud, latitud]`.

Las coordenadas representan un punto de referencia jugable, no límites administrativos ni rutas de
acceso. Los radios de descubrimiento compensan el tamaño físico de lagos, ciudades y complejos
volcánicos. No se realiza ninguna consulta a estas fuentes durante la ejecución.
